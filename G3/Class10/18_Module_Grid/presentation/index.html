Basic concepts of CSS Grid

Introduction to CSS Grid

  • Definition: CSS Grid is a two-dimensional layout system for web design.
  • Purpose: It simplifies layout creation, offering an alternative to float and flexbox-based layouts.
  • Key Features: Grid containers, grid items, rows, columns, and grid lines.

What is CSS Grid Layout?

The CSS Grid layout is a two-dimensional system that can manage both rows and columns simultaneously. Unlike flexbox, which is designed for single-dimensional layouts, CSS Grid handles layout in two directions.

Grid rows

In CSS Grid, rows are horizontal lines extending from left to right. If you're familiar with tables or spreadsheets, rows function similarly.

Grid columns

Grid columns are vertical lines extending from top to bottom. In the example below, the vertical lines represent grid columns.

Grid gap

The grid gap, also known as the gutter, is the space between rows and columns. It helps create a clean and organized layout.


.container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    grid-gap: 10px;
}

Grid lines

Grid lines are the horizontal and vertical lines that define the structure of the grid. Row lines run horizontally between rows, while column lines run vertically between columns.

Grid cell

A grid cell is the basic building block of a grid, formed by the intersection of rows and columns. It can be considered a single unit within the grid.

Grid areas

A grid area consists of one or more cells. It can be used to define specific sections of a layout, such as a header, sidebar, or footer. More details will be covered later.

Grid tracks

Grid tracks are rows or columns within the grid. For instance, all cells within a single column form a column track, while all cells within a row form a row track. Tracks can be used to define the dimensions of rows and columns.

The unit fr

The fr unit is a fractional unit used to allocate space within a grid. It divides the available space proportionally. For example, if three columns are defined with 1fr, they will each take up one-third of the available space.


.grid {
	display: grid;
	grid-gap: 10px;
	grid-template-columns: 1fr 2fr;
}

Example: The fr Unit with a 500px Width

Assume the grid is 500px wide with a 10px grid gap, leaving 490px available. If the columns are defined with 1fr + 2fr (totaling 3fr), 490px is divided proportionally.

Calculation: 1fr = 163.33px (163.33px × 3 = 490px), making 2fr equal to 326.67px.

The fr unit – 1fr = 163.333px

Now 1fr = 163.333px because 163.333 X 3 = 490 that’s it. It means 2fr = 326.667px.

Implicit vs. Explicit Grids/h2>

An explicit grid is created when rows and columns are defined using the grid-template-columns and grid-template-rows properties.


 .grid-container {
      display: grid;
      grid-gap: 10px;
      grid-template-columns: 1fr 1fr;
      grid-template-rows: 1fr 2fr;
 }
 .item {
      border: 1px solid #008000;
      padding: 20px;
 }

Grid Container (Parent Element)

A grid container is an element that wraps grid items in a box. To create a grid container, set the display property to grid.


.container {
    display: grid;
}

Grid Items (Child Elements)

Grid items are direct children of the grid container. Only direct children are considered grid items, while nested elements are not.


.container {
    display: grid;
}
.item {
    padding: 20px;
    border: 1px solid;
}

Grid Items or child elements

Grid Container Properties

The first property to set on a grid container is the display property, which converts an element into a grid container.

Grid VS Inline-Grid

The options are grid (block-level) or inline-grid (inline-level).

grid-template-columns

This property specifies the number and width of columns. A space-separated list of values defines each column's width, which can be in units like %, rem, em, px, or fr.


.container {
     display: grid;
     grid-template-columns: 1fr 2fr;
}

As we can see in the above example, the space-separated list of values is used to define the number of columns and the width of each column.

Here we can use any kind of CSS unit like %, rem, em, px, etc. to define the value of this property. Even we can use the fr unit which is enough flexible to adapt the window size or device size.

grid-template-rows

The grid-template-rows property defines the number and height of rows. It can use units such as %, rem, em, px, auto, or fr to specify each row's height.


.container {
     display: grid;
     grid-template-rows: 1fr 2fr;
}

The above example shows the space-separated list of values. These values are used to defined the number of rows and height of individual row.

We can use fr unit, auto, %, rem, em, px, etc. to define the height of each row.

repeat() function

The repeat() function helps define uniform column widths or row heights. It can be used with grid-template-columns or grid-template-rows.


.container{
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: repeat(2, 1fr);
 }

justify-content property

The justify-content property aligns grid items within the grid container horizontally.


justify-content: start | end | center | space-around | space-evenly | space-between

align-content property

The align-content property vertically aligns all grid items within the grid container. Possible values include:

  • Center
  • End
  • Start
  • Space-evenly
  • Space-around
  • Space-between


align-content: start | end | center | space-around | space-evenly | space-between

grid-column property

The grid-column property is a shorthand for grid-column-start and grid-column-end, allowing placement of an item in one or multiple columns.

Example: grid-column: 1 / span 4 places the item starting at column 1 and spanning 4 columns.


.item1 {
      grid-column: 1/4;
 }

grid-column Property

grid-column 1/ span 4


.item1 {
      grid-column: 1/ span 4;
 }

grid-row Property

Similar to grid-column, the grid-row property is a shorthand for grid-row-start and grid-row-end. It specifies the item's placement within rows.


.item2 {
   grid-row: 1/ 4;
}

grid-row property

grid rows span 4


.item2 {
    grid-row: 1/ span 4;
}

grid-template-areas

The grid-template-areas property allows placement of grid items based on named areas within the grid layout.


.grid {
	display: grid;
	grid-template-rows: 1fr 0.5fr 2fr 1fr;
	grid-template-areas: 'header header'
			     'navbar navbar'
			     'sidebar content'
			     'sidebar content'
			     'footer footer';
}
.item {
	border: 1px solid #008000;
	padding: 20px;
}

.item1 {
	grid-area: header;
}
.item2 {
	grid-area: navbar;
}
.item3 {
	grid-area: content;
}
.item4 {
	grid-area: sidebar;
}
.item5 {
	grid-area: footer;
}
